Usage

Purpose: Starts GCS get/set specific threads and then execute the following: (1) thread (x) waits for connection from a GCS get/set client (2) thread (x) sends out command request to islmgr (3) islmgr gets/sets information/commmand (4) thread (x) sends AGW Stage requested information or ERROR to client. (5) GCS client closes the connection. (6) go back to step (1)

Usage: agwServer & - Default Port = 10433 agwServer [PortNumber] &

Introduction

agwServer is a server used by the MODS AGW stage and/or TCS/IIF Actual. It accepts commands from a Client and moves mechanisms. This is for GCS interface library.

Commands

Accepts commands from a GCS client and returns information or errors.
