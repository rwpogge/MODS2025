Data Dictionary names available in the Instrument Interface (Build 2025X)

Format:
Name                             Type    Units   Description

If Name starts with 'S_' it means there are two names, one for the left side, and
one for the right side. The left side name will start with 'L_' and the right side
name with 'R_'. For example

S_MODSBlueCCDTemp                float   C       Blue camera CCD detector mount temperature

means there are two names, one for each side:

L_MODSBlueCCDTemp                float   C       Blue camera CCD detector mount temperature
R_MODSBlueCCDTemp                float   C       Blue camera CCD detector mount temperature
