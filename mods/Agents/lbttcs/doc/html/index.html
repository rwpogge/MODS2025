Usage

Usage: lbttcs [rcfile]

Where: rcfile is an optional runtime config file to load.

By default, lbttcs uses the runtime config file defined by DEFAULT_RCFILE in the client.h header.

Introduction

lbttcs is an ISIS client application that interfaces MODS data-taking system apps with the LBT Telescope Control System.

The LBT TCS uses ZeroC Ice for interprocess communication via their IIF (Instrument InterFace) facility. IIF is a subsystem of the TCS system.

lbttcs depends on two custom libraries: iifUtils/, which is the MODS IIF interface code and is the only part of the MODS data-taking system directly exposed to the IIF and ZeroC Ice, and skyUtils/ which is a suite of functions for astronomical time and coordinate conversions.

Commands

These are the interactive commands for lbttcs: 

  info           - report client information
  version        - report lbttcs version & compile info
  reset          - reset runtime & controller parameters
  verbose        - toggle verbose output mode
  debug          - toggle debugging output
  quit           - quit lbttcs
  history        - show command history
  !!             - repeat last command
  !cmd           - repeat last command matching 'cmd'
  help or ?      - view this list

Note that all commands are case-insensitive.

Runtime Configuration File

This is a typical runtime config file for the lbttcs agent: 

#
# lbttcs client runtime config file
#
# R. Pogge, OSU Astronomy Dept.
# pogge@astronomy.ohio-state.edu
# 2025 July 3 
#
################################################################

# lbttcs's ISIS client info (Host=localhost is implicit)

ID   M1.TC
Port 10801

# Application Mode: either STANDALONE or ISISclient

#Mode Standalone
Mode ISISclient

# Use with the CLI or not

#useTTY F
useTTY T

# ISIS Server Info - only releveant if Mode=ISISclient

ISISID   M1.IS
#ISISHost 192.168.139.131
ISISHost 192.168.139.92
ISISPort 6600

# Client Runtime flags 

#OpMode SIM
OpMode LIVE
Telescope LBT-SX
VERBOSE
#nolog
#debug

# ICE/IIF Instance Parameters

PropFile tcsSim
#PropFile lbtIIF
Proxy MODS1
InstID MODS
FocStation directGregorian
Side left

Note that all parameter names are case-insensitive.
